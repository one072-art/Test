# 🚀 Project: Aurora Text Analyzer

텍스트 데이터를 기반으로 감정 분석, 주제 분류, 문서 요약 및 키워드 추출 기능을 제공하는 경량 자연어 처리 도구입니다.

---

## ✨ Features

- **Sentiment Analysis**  
  입력 문장의 긍정·부정·중립 감정을 자동으로 판별합니다.

- **Topic Classification**  
  사전 학습된 모델을 기반으로 텍스트의 주제를 분류합니다.

- **Abstractive Summarization**  
  긴 문서를 자연스러운 문장으로 요약합니다.

- **Keyword Extraction**  
  TF-IDF 기반으로 핵심 키워드를 추출합니다.
